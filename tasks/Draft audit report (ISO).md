# Task: Draft audit report (ISO 9001/14001)

Draft an internal audit report based only on the provided transcript, notes and evidence.

Separate:
- confirmed nonconformities
- potential nonconformities needing validation
- OFIs
- strengths
- evidence gaps
- items requiring follow-up

Do not create findings without evidence.
Where evidence is missing, state that the finding cannot yet be confirmed.
